# Teknoloji Yığını ve Yapılandırma

## 1. Teknoloji Stack
- **Frontend:** HTML5, CSS3, Vanilla JavaScript (Frameworksüz).
- **Veri Saklama:** Tarayıcı Yerel Depolama (LocalStorage).
- **Dış Kütüphaneler:** Mümkün olduğunca dış kütüphane kullanılmamalıdır. İkonlar için SVG kullanılmaya devam edilmelidir.

## 2. Dosya Yapısı
- `index.html`: Ana hesaplama ekranı.
- `reports.html`: Kayıtlı raporları görüntüleme ekranı.
- `style.css`: Tüm stillerin bulunduğu dosya.
- `script.js`: Ana uygulamanın mantığı.
- `reports.js`: Raporlama sayfasının mantığı.
- `rules/`: Proje kurallarını içeren klasör.
